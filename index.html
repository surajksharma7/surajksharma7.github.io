Breast Cancer Prediction System

Developed a machine learning-powered web application to aid in the early detection of breast cancer. The system uses a classification model trained on tumor characteristics to predict whether a growth is benign or malignant. I built an intuitive user interface with Streamlit, enabling users to input medical measurements and receive instant, interpretable results, demonstrating a practical application of data science in healthcare.

Laptop Recommendation System

The Laptop Recommendation System is a smart, data-driven web application that helps users find laptops tailored to their needs. By analyzing technical specifications like processor, RAM, storage, and GPU, the system converts each laptop into a numerical vector and calculates similarity scores to provide accurate recommendations. Built with Python and Streamlit, it offers an interactive, user-friendly interface for exploring and selecting the best laptops quickly and efficiently.
